1. Aller sur le site Facebook Developers et connectez-vous à votre compte
  2. Allez sur la page My Apps et cliquez sur Créer une application
  3. Choisir "Consommateur" comme type d'application
  4. Renseigner un nom d'application et votre adresse email puis valider la création
  5. Ajouter a votre app le produit "Instagram Basic Display"
  6. Renseigner un nom d'application une nouvelle fois, puis cliquer sur créer
  7. Maintenant copiez les valeurs de ID de l’app Instagram et de Clé secrète de l’app Instagram dans la configuration d'Instagram2Piwigo
  8. Dans le champs URI de redirection OAuth valides, copiez-collez l'URL correpond à la règle htaccess mise en place
  9. Renseigner des url dans les champs URL d’alerte pour les annulations d’autorisation et Demandes de suppression de données et Enregistrer les modifications
  10. Cliquez ensuite sur Add or Remove Instagram Testers une nouvelle page s'ouvre
  11. Cliquez sur Ajouter Testeurs Instagram, renseignez votre non d'utilisateur instagram et validez
  12. Aller maintenant sur la page de gestion des applications et site web de votre profil instagram
  13. Dans l'onglet Invitations à tester vous devez retrouver votre application; Cliqer sur Accepter
  14. Ca devrait maintenant être Terminé!
